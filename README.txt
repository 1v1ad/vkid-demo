# VKID Demo (Next.js)

## Как запустить

1. Переименуй `.env.local.example` в `.env.local`
2. Укажи свой `VK_SERVICE_KEY`
3. Установи зависимости: `npm install`
4. Запусти проект: `npm run dev`

## Страницы
- `/login` — VKID One Tap авторизация
- `/api/vk/callback` — обработка OAuth
- `/api/auth-check` — проверка авторизации

Готово к деплою на Vercel.
